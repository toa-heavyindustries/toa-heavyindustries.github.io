Hexo常用命令整理

发表于2021-01-31|更新于2021-02-10|技术
|阅读量:

需配合Butterfly主题使用

常用参数 title:
date:
categories:
tags:
top_img:
cover:
katex:

Post Front-matter

写法 解释
title 【必需】文章标题
date 【必需】文章创建日期
updated 【可选】文章更新日期
tags 【可选】文章标签
categories 【可选】文章分类
keywords 【可选】文章关键字
description 【可选】文章描述
top_img 【可选】文章顶部图片
cover 【可选】文章缩略图
comments 【可选】显示文章评论模块(默认 true)
toc 【可选】显示文章TOC(默认为设置中toc的enable配置)
toc_number 【可选】显示toc_number(默认为设置中toc的number配置)
copyright 【可选】显示文章版权模块(默认为设置中post_copyright的enable配置)
copyright_author 【可选】文章版权模块的文章作者
copyright_author_href 【可选】文章版权模块的文章作者链接
copyright_url 【可选】文章版权模块的文章连结链接
copyright_info 【可选】文章版权模块的版权声明文字
mathjax 【可选】显示mathjax(当设置mathjax的per_page: false时，才hex需要配置，默认 false)
katex 【可选】显示katex(当设置katex的per_page: false时，才需要配置，默认 false)
aplayer 【可选】在需要的页面加载aplayer的js和css
highlight_shrink 【可选】配置代码框是否展开(true/false)(默认为设置中highlight_shrink的配置)
aside 【可选】显示侧边栏 (默认 true)

Page Front-matter

写法 解释
title 【必须】页面标题
date 【必须】页面创建日期
type 【必须】tags, categories, link
updated 【可选】页面更新日期
description 【可选】页面描述
keywords 【可选】页面关键字
comments 【可选】显示页面评论模块，默认true
top_img 【可选】页面顶部图片
mathjax 【可选】显示mathjax
katex 【可选】显示katex
aside 【可选】显示侧边栏，默认true
aplayer 【可选】在需要的页面加载aplayer的js和css
highlight_shrink 【可选】配置代码块是否默认展开

升级主题版本

升級方法：在博客根目錄下，運行 npm update hexo-theme-butterfly

图库

图库页面是普通的页面，使用hexo n page xxxxx创建页面

然后使用标签外挂galleryGroup，写法如下:

  • name - 图库名字

  • description - 图库描述

  • link - 连接到对应相册的地址

  • img-url - 图库封面的地址
